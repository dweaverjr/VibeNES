# VibeNES - Cycle-Accurate NES Emulator

A cycle-accurate Nintendo Entertainment System (NES) emulator written in C++23, designed for perfect game compatibility and hardware accuracy.

## This project is now retired. I wanted to experiment with using AI to partially vibe code an emulator. The progress was incredible, in that it works well with earlier games up to but not including games with MMC3 mappers, where the timing is complex.
## My new project is https://github.com/dweaverjr/rustendulator, where I am coding the same project but in Rust completely by hand primarily using NESDev as the main source, with no AI touching the project directly, but only in an 'ask' mode when needed for exploring ideas.
## The main reason is because I do this in my spare time, and I have found Rust to be a much more enjoyable experience than using CPP at this point.
